
# mzMusic
## *Description*
* 現在流れているBGM名を表示する
* プレイリストを選択すると、BGMを変更可能
    * 空白を選択するとBGMの一時停止/再開が可能
* ウィンドウはドラッグして移動可能

## *Commands*
* `/mu on/off` : 
	* mzMusicウィンドウのON/OFF
	* 表示/非表示を切り替えるだけなら /dn だけで可能

* `/mu reset` : 
	* mzMusicウィンドウを初期化する
	* alphaとかlayerとかいじっておかしくなったらresetしてね

* `/mu alpha [数値]` : 
	* [数値]は0～100の範囲で指定する
	* ウィンドウ背景の透過率が設定できる
	* 初期値は70で、0に近いほど透明になる

* `/mu layer [数値]` : 
	* [数値]は0～100の範囲で指定する
	* ウィンドウのレイヤ階層が設定できる
	* 初期値は70で、0に近いほど後ろになる
	* 普段使う分には多分いらない機能
